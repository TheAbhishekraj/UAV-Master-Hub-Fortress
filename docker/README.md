# 🐳 DOCKER SILO
**The Environment of Reproducibility**

This silo contains the containerization logic that guarantees the "Golden Vault" reproducibility of the research ecosystem.

## 📁 Key Components
- **`Dockerfile.golden`**: The immutable image specification.
- **`docker-compose.golden.yml`**: The multi-service orchestration for Gazebo, PX4, and ROS 2.
- **`scripts/`**: Automation for display forwarding and container initialization.
